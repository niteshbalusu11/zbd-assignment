### Running the App



### Using docker
```
# Clone the repo

git clone https://github.com/niteshbalusu11/zbd-assignment.git

cd zbd-assignment

# Pass GF_SECURITY_ADMIN_USER and GF_SECURITY_ADMIN_PASSWORD as env variables
# For authenticating into grafana

docker compose up -d

# Access grafana dashboard on
localhost:3001
```
